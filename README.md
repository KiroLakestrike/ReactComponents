## Components

### Social Links

Social Links is from a HTML Javascript Tutorial on Youtube that i decided to port into React.
[Link to the Original Video](https://www.youtube.com/watch?v=UQKWc2r_41U) 

How to use in your Project:
The useEffect will fetch the social Links from your backend.
You can also use the socials.json file in the public folder.

Implement with <SocialLinks />
